# 🐾 WanCos Pet - Tienda Virtual

¡Bienvenido a la tienda oficial de **WanCos Pet**! Un espacio dedicado a consentir a tus peludos con los mejores accesorios y juguetes.

## 🚀 Características
* **Catálogo Interactivo:** Separado por categorías (Perros y Gatos).
* **Carrito de Compras:** Cálculo automático de Subtotal e IVA (19%).
* **Galería Premium:** Visualización de productos con zoom mediante Fancybox.
* **Simulador de Pagos:** Cálculo de costos de envío según la ubicación.
* **Pedido vía WhatsApp:** Finalización de compra directa y personalizada.

## 🛠️ Tecnologías utilizadas
* **HTML5 / CSS3:** Diseño responsive y moderno.
* **JavaScript (Vanilla):** Lógica del carrito y persistencia en LocalStorage.
* **Fancybox:** Para una experiencia visual fluida.
* **Google Fonts:** Tipografía "Poppins" para legibilidad.

## 📦 Instalación y Uso
1. Clona el repositorio.
2. Asegúrate de tener las carpetas `img_sitio` e `img_tienda` con sus respectivos recursos.
3. Abre `index.html` en tu navegador.
